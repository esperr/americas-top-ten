About America's Top Ten Bills

Why America's Top Ten Bills?

America's Top Ten Bills is a remixing of the data found at the 10 Most Viewed Bills page. It republishes the weekly list found there while enhancing it is a couple of important ways. Firstly, it shows the sponsor of each measure. Second, it not only shows the current rank of each measure, but also past its past performance, measured by previous week's rank, peak rank and total number of weeks on the chart.

Technology and Credits

All data comes from the friendly folks at congress.gov. There's not presently an API for extractng this information, so this project rests on an edifice of screen-scraping using beautifulsoup. See this repo for details
